# 扩展概述

::: tip
Yiitap 提供了 15+ 自定义扩展，基于这些扩展和Tiptap提供的扩展，Yiitap可以帮助实现一个特性丰富的编辑器。
:::

## 自定义扩展
- Callout
- Char Command
- Code Block
- Color Highlighter
- Image
- Placeholder
- Table
- Trailing Node
- Unique ID
- Video

## Tiptap扩展
Yiitap集成了Tiptap大部分扩展：
- BackColor
- [Blockquote](https://tiptap.dev/docs/editor/extensions/nodes/blockquote)
- Bold
- BulletList
- Code
- FontFamily
- ForeColor
